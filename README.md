# CyberBot - Cybersecurity Awareness Chatbot

A C# console chatbot that educates users about cybersecurity threats and best practices.

# Features
-  Voice greeting on startup
-  ASCII art logo with coloured console UI
-  Personalised responses using your name
-  Cybersecurity Q&A (passwords, phishing, malware, VPN, 2FA, ransomware & more)
-  Input validation for empty/unknown input
-  Typing animation effect

# Topics You Can Ask About
- Passwords
- Phishing
- Malware / Viruses
- VPN
- Firewall
- Two-Factor Authentication (2FA)
- Ransomware
- Social Engineering
- Safe Browsing
- Software Updates
